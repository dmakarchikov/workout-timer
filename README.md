# Таймер упражнений 💪

Приложение для выполнения реабилитационных и силовых тренировок с таймером подходов и отдыха.

🔗 **[Открыть приложение](https://dmakarchikov.github.io/workout-timer/)**

---

## Возможности

- **Два типа упражнений:**
  - Повторы без таймера — нажми «Сделал» после каждого подхода, приложение запустит отдых
  - Удержание с таймером — таймер стартует автоматически, подходы идут без паузы
- **Звуковые сигналы:**
  - Сигнал по окончании упражнения
  - Предупреждение за 5 секунд до конца отдыха (тип 2)
  - Три восходящих бипа — старт следующего подхода
  - Финальный аккорд по завершении тренировки
- **Прогресс:** прогресс-бар и индикатор подходов
- **Офлайн:** работает без интернета после первого открытия

---

## Установка на телефон (Android)

1. Открой [ссылку](https://dmakarchikov.github.io/workout-timer/) в **Chrome**
2. Меню ⋮ → **Добавить на главный экран**
3. Приложение установится как нативное — без адресной строки

На iOS: Safari → кнопка «Поделиться» → **На экран «Домой»**

## Тренировки

Тренировки хранятся в массиве `workouts` в `index.html`. Формат упражнения:

```js
// Тип 1 — повторы без таймера
{
  type: 1,
  title: "Название",
  description: "Описание техники",
  sets: 1,          // количество подходов
  reps: 20,         // повторов за подход
  restSeconds: 30   // отдых между подходами
}

// Тип 2 — удержание с таймером (автостарт)
{
  type: 2,
  title: "Название",
  description: "Описание техники",
  sets: 5,          // количество подходов
  holdSeconds: 20,  // секунд удержания за подход
  restSeconds: 25   // отдых между подходами
}
```
